# FULL_ACTUAL_COMPACT companion note [примечание о полной актуальной компактной версии]

Public Release [публичный релиз] v1.4.10 is accompanied by a separate FULL_ACTUAL_COMPACT [полная актуальная компактная версия] for the user.

The companion archive preserves active memory [активную память], restore prompts [промпты восстановления], legal code registration text content [текстовое содержание для регистрации кода], QA reports [отчёты проверки], rule maps [карты правил], active core references [ссылки на активное ядро], and history manifests [манифесты истории].

It does not embed old full binary ZIP archives [старые полные бинарные ZIP-архивы]. Historical continuity [историческая преемственность] is preserved through manifest [манифест], SHA-256 [контрольная сумма SHA-256], size [размер], role [роль], digest cards [карточки-сводки], and local/chat-history storage [локальное хранение / истории чатов].
