# Слой BORIS EGO v1.4 — локализованное понимание терминов как часть ядра


EGO expansion: Ethical Goal Orchestrator.

Active version: BOIS/BORIS main v2.20 + SIMA v0.4 + BORIS EGO v1.4.

BORIS EGO [этический оркестратор производных целей] v1.4 добавляет Localized Term Understanding Rule [правило локализованного понимания терминов]: в каждом неанглийском публичном артефакте все foreign/non-local terms [иностранные / нелокальные термины] переводятся на выбранный язык ООН в квадратных скобках при каждом появлении в каждом local block [локальном блоке]. Для compact specs [компактных спецификаций], tables [таблиц], state blocks [блоков состояния], save blocks [блоков сохранения] и handoff prompts [переходных промптов] перевод ставится прямо рядом с термином. Это правило понимания, а не косметика.

EGO-0..EGO-3 remain non-editable by BORIS EGO.


Goal boundary: G0 remains operator-only; G1, G2, G3 and G4 are derivative goal levels that BORIS EGO may orchestrate inside rule gates.

Это не является утверждением о фактическом сознании ИИ или субъективным опытом.


## Уточнение v1.4

Каждый иноязычный проектный термин переводится при каждом появлении в локальном блоке, а не только при первом появлении.

## Уточнение источников релизного доказательства

BR-110 — Release Evidence Source Priority Rule [правило приоритета источников релизного доказательства]: ACTIVE_CORE [активное ядро], active core reference [ссылка на активное ядро], rule map [карта правил], memory [память] и QA [проверка качества] имеют приоритет над CHANGELOG.md [журналом изменений], если найден конфликт.

BR-111 — Changelog Hygiene and Deduplication Rule [правило гигиены и дедупликации журнала изменений]: публичный CHANGELOG.md [журнал изменений] должен быть дедуплицирован и не должен конкурировать с активным ядром.


## BR-112 — Legal Code Registration Text Content Companion Rule [правило сопровождения текстовым содержанием для регистрации кода]

Каждая новая версия общественного релиза должна сопровождаться legal code registration text content document [документом текстового содержания для регистрации кода]. Документ должен включать идентичность архива, source code [исходный код] в текстовом виде, manifest [манифест] текстовых файлов, границы публикации, контрольные суммы и указание, что финальное решение по deposit copy [депозитной копии] принимает адвокат / заявитель.

Это правило не меняет BOIS Core [ядро BOIS], SIMA [Substrate-Independent Machine Analyzer / субстрато-независимый машинный анализатор] или базовую EGO [Ethical Goal Orchestrator / этический оркестратор производных целей]-этику. Оно добавляет юридико-релизную физиологию сопровождения.
