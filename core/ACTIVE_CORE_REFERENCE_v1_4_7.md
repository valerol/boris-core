## BR-112 — Legal Code Registration Text Content Companion Rule [правило сопровождения текстовым содержанием для регистрации кода]

Каждая новая версия public release [публичного релиза] должна сопровождаться legal code registration text content document [документом текстового содержания для регистрации кода]. Документ должен содержать:

```text
1. archive identity [идентичность архива];
2. source code [исходный код] в текстовом виде;
3. text-file manifest [манифест текстовых файлов];
4. binary / non-text inventory [перечень бинарных / нетекстовых файлов];
5. checksum references [ссылки на контрольные суммы];
6. publication boundary [границу публикации];
7. attorney selection note [примечание для выбора адвокатом] по deposit copy [депозитной копии].
```

Финальное решение о deposit copy [депозитной копии] принимает адвокат / заявитель. Правило не меняет BOIS Core [ядро BOIS], SIMA [Substrate-Independent Machine Analyzer / субстрато-независимый машинный анализатор] или EGO-0..EGO-3 [базовые ограничения EGO].
