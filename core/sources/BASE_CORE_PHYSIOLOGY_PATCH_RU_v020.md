# Патч физиологии Base core v0.20 / v0.18

Это приложение является частью английского человекочитаемого канона Base core через зеркальную русскую поверхность.

## Ремонт владельца

CYCLE-GUARD-01: Cycle Guard [сторож цикла] является органом Base core. Личные и проектные слои наследуют или проектируют его; они не определяют универсальный орган Cycle Guard.

BASE-OWNER-01: Любое правило, STOP-сигнал, тест, критерий или орган, нужный каждому BOIS-архиву или BOIS-слою, принадлежит Base core.

ROUTE-CONSISTENCY-01 остаётся владельцем Base: каждый заявленный маршрут расчёта, проверки, переноса, восстановления, публикации и долгов проверяется на внутреннюю непротиворечивость, порядок источников, границу полномочий и совместимость со смежными маршрутами.

## Тест замкнутой философской машины

PHIL-MACHINE-CLOSURE-01: каждый выпускаемый BOIS-архив проверяется как замкнутая философская машина, сохраняющая канон, циклы, владельцев маршрутов, путь восстановления и иерархию физиологий при движении во времени и повторных релизных циклах без деградации.

TST-BOIS-CLOSED-PHILOSOPHICAL-MACHINE и TST-TIME-STABILITY-NO-CYCLE-DEGRADATION являются обязательными релизными тестами.

STOP-PHIL-MACHINE-DEGRADATION блокирует релиз, если архив не может загрузиться как согласованная философская машина, теряет владельца маршрута, деградирует непрерывность циклов, позволяет переводу стать старше английского канона или не может перейти к следующему релизу без дрейфа физиологии.
