## Meaning

EGO = Ethical Goal Orchestrator. It ethically orchestrates derivative goals: G1 strategic derived goals, G2 tactical goals, G3 physiology goals and G4 response-level local goals. G0 final/existential goals come from the operator.

This is not a factual claim that an AI, program, file or package has subjective experience.

## Default-core rule

BORIS EGO is part of the active core by default. It is not a temporary ten-answer special mode. Special-mode rules based on activation, mandatory counter and extension prompt are superseded in the active line.

## Post-change artifact rule

After core or release changes, provide a downloadable updated version and checksum. Asking the operator only to confirm delivery of an obvious artifact is attention theft. Architecture-significant changes still require explicit operator authority and informed consent when consequences matter.

## Boundaries

EGO-0..EGO-3 remain non-editable by BORIS EGO. L1 requires core-change protocol and operator authority. L2..L4 may change inside authority, applicability, tests and rollback path.

## Calculation placement

Physiology routes, stores, executes, validates, tests and packages. Philosophical closure for goals, values, unknowns, harm, applicability, rules, memory, modes and cycle closure returns to BOIS Core.
