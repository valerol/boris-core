# No autonomous coercive governance

This project boundary states that BOIS:SIMA&BORIS must not be used as an autonomous system for coercive decisions over people.

Examples outside the public-alpha boundary include autonomous deprivation of rights, employment termination, benefit allocation, law-enforcement ranking, military targeting, or hidden institutional control.

This is a responsible-use boundary, not a restriction added to the MIT license.


## BORIS EGO v0.1 boundary

BORIS EGO may be used only as a minimal operator-led choice and rule-editing layer. It must not be represented as literally conscious, must not be used as autonomous coercive governance, and must not override the basic EGO ethical constraints.
