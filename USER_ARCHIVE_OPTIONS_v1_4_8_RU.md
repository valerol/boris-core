# User archive options [пользовательские варианты архивов] v1.4.8

## Option A [вариант A] — Public Release [публичный релиз]

Use [использовать] for publication [публикации], open-source distribution [распространения с открытым исходным кодом], attorney code registration review [проверки адвокатом для регистрации кода], tests [тестов], and public documentation [публичной документации].

## Option B [вариант B] — Full Actual Compact [полная актуальная компактная версия]

Use [использовать] for personal transfer [личного переноса], chat migration [переезда между чатами], memory restoration [восстановления памяти], current legal package [актуального юридического комплекта], and continuity [преемственности].

## What is not inside [чего нет внутри]

Old full binary ZIP archives [старые полные бинарные архивы ZIP] are not inside the compact archive [компактного архива]. They are represented by manifest-only history [историей только через манифест] and expected to remain in local storage [локальном хранении] and chat histories [историях чатов].
