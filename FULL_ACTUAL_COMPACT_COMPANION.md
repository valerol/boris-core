# Full Actual Compact Companion [сопровождение полной актуальной компактной версии]

Public Release [публичный релиз] v1.4.10 is paired with Full Actual Compact [полная актуальная компактная версия] v1.4.10.

The Public Release [публичный релиз] contains open publication materials [материалы открытой публикации], code [код], documentation [документация], tests [тесты] and legal code registration text content [текстовое содержание для регистрации кода].

The Full Actual Compact [полная актуальная компактная версия] contains current memory [текущая память], restore resources [ресурсы восстановления], transition package [переходный пакет], legal companion [юридическое сопровождение], QA reports [отчёты проверки] and manifest-only historical continuity [историческая преемственность только через манифест].

Historical binary archives [исторические бинарные архивы] are stored locally and in chat histories [историях чатов]. They are not embedded in the working compact archive [рабочий компактный архив].
