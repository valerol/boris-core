# Governance

This anonymized public alpha is published as a static archive. Product continuation is paused until a team, public repository process, and security triage process exist.

Maintainer identity is intentionally not included in this archive. A future repository may add maintainers, security contacts, CI, tags, and issue triage.

No component may autonomously change goals, rules, deployment boundaries, or decisions about people. Changes require human review, tests, changelog, and rollback path.
