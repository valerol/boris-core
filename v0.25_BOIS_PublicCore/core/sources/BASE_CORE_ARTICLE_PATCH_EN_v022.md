# Base core article patch v0.22 - Socrates article strict philosophical machine addendum

PHIL-MACHINE-DEF-01 was already inserted into Base core v0.22. v0.22 aligns the non-excludable Socrates / BOIS article with that strict definition.

The article now distinguishes structures, organs, opers and operoms from philosophical machines. A philosophical machine is only a time-looped construction that consumes an input flow on a substrate, produces philosophical categories, returns them into the substrate, accumulates changes and can reach phase transition.

No new organ is introduced by this patch.
