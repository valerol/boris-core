# Патч статьи Base core v0.22 - дополнение к статье о Сократе

PHIL-MACHINE-DEF-01 уже внесено в Base core v0.22. v0.22 согласует обязательную статью о Сократе / BOIS с этим строгим определением.

Статья теперь различает структуры, органы, оперы и оперомы с философскими машинами. Философская машина - только зацикленная во времени конструкция, которая потребляет входной поток на субстрате, производит философские категории, возвращает их в субстрат, накапливает изменения и может доходить до фазового перехода.

Новый орган этим патчем не вводится.
