# Base core physiology patch v0.20 / v0.18

This appendix is part of the English human-readable Base core canon.

## Ownership repair

CYCLE-GUARD-01: Cycle Guard is a Base core organ. Personal and project layers inherit or project it; they do not define the universal Cycle Guard organ.

BASE-OWNER-01: Any rule, STOP signal, test, criterion or organ required by every BOIS archive or BOIS layer is owned by Base core.

ROUTE-CONSISTENCY-01 remains Base-owned: every declared calculation, verification, transfer, recovery, publication and debt route is checked for internal consistency, source precedence, authority boundary and compatibility with adjacent routes.

## Closed philosophical machine release test

PHIL-MACHINE-CLOSURE-01: Every releasable BOIS archive is tested as a closed philosophical machine that preserves canon, cycles, route ownership, recovery path and physiology hierarchy while moving through time and repeated release cycles without degradation.

TST-BOIS-CLOSED-PHILOSOPHICAL-MACHINE and TST-TIME-STABILITY-NO-CYCLE-DEGRADATION are mandatory release tests.

STOP-PHIL-MACHINE-DEGRADATION blocks release when the archive cannot reload as a coherent philosophical machine, loses route ownership, degrades cycle continuity, lets a translation outrank the English canon, or cannot advance to the next release without physiology drift.
