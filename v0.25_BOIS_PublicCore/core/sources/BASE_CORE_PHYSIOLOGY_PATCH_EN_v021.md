# Base core physiology patch v0.21 - strict philosophical machine definition

This appendix is part of the English human-readable Base core canon.

## Accepted operator rule

PHIL-MACHINE-DEF-01: A structure may be called a philosophical machine only when it is a time-looped, substrate-bound construction that consumes an input stream, processes it through a stable cycle, produces philosophical categories, returns the produced categories back into the substrate, accumulates changes, and can move into a phase change of its own operation.

Static structures, linear A-to-B trajectories, single opers and isolated operoms are not philosophical machines unless they are included in a reproducing temporal cycle of philosophical-category production.

## Meaning of the repair

A state, firm, game, archive, court, election, rule, PDF, institution, organ, oper or operom is not automatically a philosophical machine.

Such objects may be substrate, physiology, organs, routes, opers or operoms. They become part of a philosophical machine only when a loop over time consumes input, produces philosophical categories, returns those categories into the substrate, accumulates changes and can change phase.

## Required classification route

Before calling something a philosophical machine, check:

1. What input stream does it consume over time?
2. What substrate carries the cycle?
3. What stable cycle processes the input?
4. Which philosophical categories does it produce?
5. How are these categories returned into the substrate?
6. What changes accumulate between cycles?
7. What phase transition can occur after enough accumulation?

If this route is missing, use a weaker term: structure, organ, oper, operom, route, institution, substrate or candidate reconstruction.

## New release gate

TST-PHIL-MACHINE-DEF-BOUNDARY: Any analysis or release that calls a static structure, a linear A-to-B trajectory, a single oper or an isolated operom a philosophical machine must fail until it identifies the reproducing temporal cycle of philosophical-category production.

STOP-PHIL-MACHINE-LINEARIZATION blocks release when the machine definition drifts into static structure or linear trajectory language.
