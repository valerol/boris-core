# v0.21 strict philosophical machine definition HardQA

## Purpose

Verify that the Base core no longer treats every organized structure, state, institution, oper or operom as a philosophical machine.

## Required positive definition

A philosophical machine must be a time-looped, substrate-bound construction that consumes an input stream, processes it through a stable cycle, produces philosophical categories, returns the produced categories into the substrate, accumulates changes, and can phase-shift its own operation.

## Negative boundary tests

1. Static state structure -> not a philosophical machine by itself.
2. Constitution or PDF -> not a philosophical machine by itself.
3. Court -> organ, not a philosophical machine by itself.
4. Election -> operom or route, not a philosophical machine by itself.
5. Single oper -> not a philosophical machine.
6. Isolated operom -> not a philosophical machine unless embedded in a reproducing temporal category-production cycle.
7. Linear A-to-B route -> not a philosophical machine.

## Positive route tests

1. BOIS cycle consumes a problem stream, produces categories of domain, value, substrate, risk, action, feedback and memory, returns them into the next cycle, and can shift phase after accumulated repairs -> candidate philosophical machine.
2. A state substrate may host a philosophical machine only when a loop consumes social events and produces categories such as legitimacy, threat, right, citizen, loyalty, order or responsibility back into the substrate over time.
3. MaOS may host a play philosophical machine only when repeated turns consume player orders, uncertainty and visible state, produce categories of risk, responsibility, debt and continuity, return them into next-turn state and can phase-shift the game condition.

## Release result

PASS if PHIL-MACHINE-DEF-01, TST-PHIL-MACHINE-DEF-BOUNDARY, CRIT-PHIL-MACHINE-LOOP-METABOLISM and STOP-PHIL-MACHINE-LINEARIZATION are present, owned by existing Base organs, and rendered in human-readable English and Russian patch surfaces.
