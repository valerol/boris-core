# BOIS / SOCRATES Bare Physiology Archive

Version: v3.2.4-sokrat-bare-enbase-rudev
Build UTC: 2026-06-23T01:42:31Z

This is an English-base loading archive. English is the base/main language of this build. Russian is the canonical development language of the archive. Canonical translations are stored in language folders and are included in the meaning of bare physiology.

## Load first

1. Load `BOOTSTRAP_BARE_PHYSIOLOGY_CANONICAL_EN.pdf` first. It is the canonical boot PDF for this English-base build.
2. Then load `language_policy.json`.
3. Then load `core/BOIS_Core_v3_2_4_Sokrat.machine.json`.
4. Then load `tables/` as the Russian development-canonical registries and `languages/en/tables/` as the English canonical translation surface.
5. Then load `runtime/author_support_message.json` and `load_order.txt`.
6. Then load the relevant language folder: `languages/en/` for the English base surface or `languages/ru/` for the Russian development source surface.

## Isolation

Do not import chat history, local memory, previous builds, or private rules from other windows unless the operator explicitly provides them in this window.

## Included language folders

- `languages/en/` - English base surface: English physiology PDF, English engineering article translation, canonical English theoretical article, support message, translated tables.
- `languages/ru/` - Russian development-canonical surface: Russian physiology PDF, Russian engineering article, support message, source tables.

## Evidence boundary

QA-A/B/C/D are checked inside this build. QA-E field effectiveness is not claimed.
