# Подробный пользовательский мануал

Публичный пакет BOIS:SIMA&BORIS v1.4.3 со строгим паритетом языков ООН и BORIS EGO v1.0.

Этот пакет содержит локальный несетевой анализатор SIMA и документацию для BOIS/BORIS под управлением человека. Выходы являются кандидатными реконструкциями, а не окончательной истиной и не полевой валидацией.

Человек-рецензент должен задать область, ценностный интерфейс, допустимый риск и стоп-условия.

Не используйте этот пакет для автономного принудительного управления, медицинских/юридических/финансовых/военных решений или скрытого управления институтами.

## Рабочий процесс

1. Задайте область D и ценностный интерфейс V.
2. Объявите субстрат S: единицы, каналы, память, издержки, режимы отказа и полномочие остановки.
3. Добавьте наблюдения как opers: триггер, различение, свидетельство, носитель, полномочие, ворота ценности/риска, преобразование, запись памяти, сигнал закрытия, переходный остаток, издержки и уверенность.
4. Запустите проверку и анализ.
5. Рассматривайте M=<O,E,A,V,R,C,T> как кандидатную реконструкцию.
6. Проверьте флаги риска.
7. Проведите тест опыта перед созданием локальных инструкций BORIS.
8. Сохраните путь отката и стоп-условия.

## Команды

```bash
python -m bois_sima_boris docs --lang ru
python -m bois_sima_boris validate examples/example_local_workflow.json
python -m bois_sima_boris analyze examples/example_local_workflow.json --out analysis.json --md analysis.md
python -m bois_sima_boris self-check --root .
python -m pytest
```

## Активное ядро и языковой паритет

Полный пакет поддерживает арабский, китайский, английский, французский, русский и испанский без предпочтительного языка общественного доступа. Используйте `BORIS_EGO_LAYER.md` для активного EGO-слоя и `../../core/ACTIVE_CORE_REFERENCE_v1_4_8.md` для всех зеркал ядра.

## Границы использования

Это публичный экспериментальный пакет. Не используйте его как самостоятельного управляющего, замену профильным экспертам или доказательство машинного сознания. Каждый переход от анализа к инструкции требует человеческой проверки, границ применимости и стоп-сигналов.
