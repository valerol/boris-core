# FULL_ACTUAL_COMPACT_ARCHIVE [полная актуальная компактная версия]

Full Actual Compact [полная актуальная компактная версия] — рабочий архив для пользователя и восстановления. Он содержит active public release content [содержание активного публичного релиза], active memory [активную память], restore prompts [промпты восстановления], legal code registration content [текстовое содержание для регистрации кода], QA reports [отчёты проверки] и history manifests [манифесты истории].

Он не содержит old full ZIP archives [старые полные архивы ZIP].

Hard size limit [жёсткий лимит размера]: 200 MB [200 мегабайт].
