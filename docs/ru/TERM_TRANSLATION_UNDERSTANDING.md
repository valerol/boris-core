# Слой BORIS EGO v1.3 — локализованное понимание терминов как часть ядра

BORIS EGO [этический оркестратор производных целей] v1.3 добавляет Localized Term Understanding Rule [правило локализованного понимания терминов]: в каждом неанглийском публичном артефакте все foreign/non-local terms [иностранные / нелокальные термины] переводятся на выбранный язык ООН в квадратных скобках при каждом появлении в каждом local block [локальном блоке]. Для compact specs [компактных спецификаций], tables [таблиц], state blocks [блоков состояния], save blocks [блоков сохранения] и handoff prompts [переходных промптов] перевод ставится прямо рядом с термином. Это правило понимания, а не косметика.
