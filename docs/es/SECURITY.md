# Seguridad

Archivo alfa público final anonimizado

Este paquete localizado proporciona un analizador SIMA local y sin red, más documentación para el uso humano de BOIS/BORIS. Produce reconstrucciones candidatas, no verdad final ni validación de campo.

Una persona revisora debe definir dominio, interfaz de valores, riesgo aceptable y condiciones de parada.

No use este paquete alfa para gobernanza coercitiva autónoma, decisiones médicas/jurídicas/financieras/militares ni control institucional oculto.

Lanzamiento soportado: v1.3.9 alpha público con paridad de idiomas de la ONU y BORIS EGO v0.6.

Use el reporte privado de vulnerabilidades del repositorio después de que exista un repositorio público. Este archivo con artículo incluido no contiene correo personal de respaldo para seguridad; la identidad del autor aparece solo en los archivos de artículo incluidos. No publique detalles de explotación.

Base de seguridad: sin red, sin ejecución shell, sin ejecución dinámica y sin gobernanza autónoma.
