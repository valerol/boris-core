# Stabilité du format de réponse

Active release: v1.4.4; BORIS EGO v1.1.

The default answer envelope is `MAXIMUM_OPERATIONAL_RESPONSE`. It preserves the answer counter, active-core reference, summary-first structure, separation of facts/assumptions/conclusions/risks, test status, artifact links after changes, and limits of certainty.

This rule fixes cross-chat and in-chat format drift. The envelope may compress repeated history, but must not delete active context or viability information.
