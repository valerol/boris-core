# Neutralité linguistique

Cette version prend en charge l’arabe, le chinois, l’anglais, le français, le russe et l’espagnol comme langues d’entrée publique égales. Aucune langue naturelle n’est préférée.

L’ordre des langues dans les fichiers, les listes ou les noms n’indique pas une priorité. Si un conflit sémantique apparaît entre traductions, il doit être résolu par les invariants BOIS, les cartes de règles, l’intention explicite de l’opérateur, les limites d’applicabilité, les tests et les chemins de retour arrière.

Les traductions assistées par machine ou non certifiées doivent être traitées comme texte révisable, non comme formulation juridique certifiée.
