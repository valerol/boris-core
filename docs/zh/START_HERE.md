带有联合国语言平等和 BORIS EGO v1.0 的公共 alpha 包。

没有任何公共访问语言被设为优先。索引中的语言顺序只是固定顺序，不表示优先级。

快速开始：先读本文件，再读 `QUICKSTART.md` 和 `USER_MANUAL_SHORT.md`。完整使用请读 `USER_MANUAL_FULL.md` 和 `BORIS_EGO_LAYER.md`。
