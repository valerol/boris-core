# BORIS EGO v1.3 层 — 作为默认核心的本地化术语理解

BORIS EGO [伦理性派生目标编排器] v1.3 添加 Localized Term Understanding Rule [本地化术语理解规则]：在每个非英语公共制品中，每个 foreign/non-local term [外来/非本地术语] 在每个 local block [本地块] 中第一次出现时，必须用方括号给出所选联合国语言的译文。在 compact specs [紧凑规格], tables [表格], state blocks [状态块], save blocks [保存块] 和 handoff prompts [交接提示] 中，译文必须直接放在术语旁边。这是理解规则，不是装饰性格式规则。
