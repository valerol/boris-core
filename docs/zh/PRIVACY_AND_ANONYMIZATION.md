# 隐私与匿名化

匿名化最终公共 Alpha 档案

此本地化包提供仅本地运行、无网络的 SIMA 分析器，以及由人类主导使用 BOIS/BORIS 的文档。它产生候选重构，不是最终真理，也不是现场验证。

人类审查者必须设定领域、价值接口、可接受风险和停止条件。

不要将此 Alpha 包用于自主强制治理、医疗/法律/金融/军事决策或隐藏的机构控制。

This package keeps code and technical documentation generic. Author identity and correspondence metadata appear only inside bundled article files.

If you add examples, remove private names, addresses, emails, project codenames, proprietary business plans, client data, and internal logs before publication.
