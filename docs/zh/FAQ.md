# 常见问题

匿名化最终公共 Alpha 档案

此本地化包提供仅本地运行、无网络的 SIMA 分析器，以及由人类主导使用 BOIS/BORIS 的文档。它产生候选重构，不是最终真理，也不是现场验证。

人类审查者必须设定领域、价值接口、可接受风险和停止条件。

不要将此 Alpha 包用于自主强制治理、医疗/法律/金融/军事决策或隐藏的机构控制。

问：这是自主代理吗？答：不是。
问：它使用互联网吗？答：不使用。
问：它能证明对象的真实哲学机器吗？答：不能，它只重构候选项。
问：它能构建本地 BORIS 系统吗？答：它可以指导由人类主导的本地 BORIS 设计，但必须进行经验测试。
问：是否包含私人项目材料？答：没有有意包含私人项目材料。作者身份只出现在捆绑的文章文件中。
