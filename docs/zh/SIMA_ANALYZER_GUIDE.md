# SIMA 分析器指南

匿名化最终公共 Alpha 档案

此本地化包提供仅本地运行、无网络的 SIMA 分析器，以及由人类主导使用 BOIS/BORIS 的文档。它产生候选重构，不是最终真理，也不是现场验证。

人类审查者必须设定领域、价值接口、可接受风险和停止条件。

不要将此 Alpha 包用于自主强制治理、医疗/法律/金融/军事决策或隐藏的机构控制。

SIMA reconstructs candidate philosophical machines from observed substrate physiology. It does not prove the true machine of a real object.

Output includes substrate declaration, value interface, operome, candidate machines, risk flags, limits, and next step.
