# 安全

匿名化最终公共 Alpha 档案

此本地化包提供仅本地运行、无网络的 SIMA 分析器，以及由人类主导使用 BOIS/BORIS 的文档。它产生候选重构，不是最终真理，也不是现场验证。

人类审查者必须设定领域、价值接口、可接受风险和停止条件。

不要将此 Alpha 包用于自主强制治理、医疗/法律/金融/军事决策或隐藏的机构控制。

受支持的发布版：v1.3.9，具有联合国语言平等和 BORIS EGO v0.6 的公共 alpha。

创建公共代码库后，请使用代码库的私密漏洞报告渠道。本含文章档案不包含个人安全备用邮箱；作者身份只出现在捆绑的文章文件中。不要公开发布漏洞利用细节。

安全基线：无网络、无 shell 执行、无动态执行、无自主治理。
