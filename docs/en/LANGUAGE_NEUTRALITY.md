# Language neutrality

This release supports Arabic, Chinese, English, French, Russian, and Spanish as equal public-entry languages. No natural language is preferred.

Language order in files, lists, or names does not indicate priority. If semantic conflict appears between translations, resolve it through BOIS invariants, rule maps, explicit operator intent, applicability limits, tests, and rollback paths.

Machine-assisted or non-certified translations must be treated as revisable text, not certified legal wording.
