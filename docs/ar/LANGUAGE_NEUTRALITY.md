# الحياد اللغوي

يدعم هذا الإصدار العربية والصينية والإنجليزية والفرنسية والروسية والإسبانية كلغات دخول عامة متساوية. لا توجد لغة طبيعية مفضلة.

ترتيب اللغات في الملفات أو القوائم أو الأسماء لا يعني أولوية. إذا ظهر تعارض دلالي بين الترجمات، يُحسم عبر ثوابت BOIS، وخريطة القواعد، ونية المشغّل الصريحة، وحدود التطبيق، والاختبارات، ومسار التراجع.

المحتوى غير المراجع بشرياً في الترجمة يجب أن يعامل كمحتوى قابل للمراجعة، لا كصياغة قانونية معتمدة.
