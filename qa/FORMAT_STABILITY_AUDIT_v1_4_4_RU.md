# Response format stability audit — v1.4.4

Default format: MAXIMUM_OPERATIONAL_RESPONSE.

Stable envelope required by default:

1. Answer counter.
2. Active core reference.
3. Summary-first report.
4. Assumptions / facts / conclusions / risks separation.
5. Test and verification status.
6. Limits of certainty.
7. Artifact links after changes.
8. Memory / transition resources at cycle closure.

This gate repairs recurrent format drift during chat-to-chat transfer and inside long chats.
