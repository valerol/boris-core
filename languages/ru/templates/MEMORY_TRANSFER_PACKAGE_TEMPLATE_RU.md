# Шаблон переноса памяти

Используется при сообщении оператора #50 и после #50, если оператор продолжает работу.

## Обязательные блоки
- версия активной физиологии;
- D/V/S;
- принятые решения;
- артефакты и ссылки;
- открытые долги;
- активные риски и STOP-сигналы;
- rollback/fallback [откат/резервный путь];
- инструкция загрузки в следующем чате;
- state_delta [изменение состояния].
