# BOIS / SOCRATES - загрузочный архив голой физиологии

Эта сборка имеет английский базовый язык. Русский остаётся каноническим языком разработки архива и источником переводов.

## Порядок загрузки
1. Сначала загрузить `../../BOOTSTRAP_BARE_PHYSIOLOGY_CANONICAL_EN.pdf` - канонический загрузочный PDF этой английской сборки.
2. Затем `../../language_policy.json`.
3. Затем `../../core/BOIS_Core_v3_2_4_Sokrat.machine.json`.
4. Затем русские канонические таблицы разработки из `tables/` и английские переводы из `../en/tables/`.
5. Затем `../../runtime/author_support_message.json`.

## Граница
QA-A/B/C/D проверяются статически; QA-E [полевая эффективность] не заявляется.

## Ремонт v0.17

Эта сборка ремонтирует полноту переноса, предполётную проверку Cycle Guard [сторожа цикла], авторское сообщение поддержки для скачиваемых BOIS-продуктов, границы голого архива и вопросы к оператору до существенного расчёта. ## Ремонт v0.24: авторское сообщение поддержки

SUPPORT-MESSAGE-ANONYMIZATION-IMMUNITY-01. Авторское сообщение поддержки является неисключаемой частью ядра BOIS и видимой поверхности вывода. Анонимизация может удалять приватную оперативную память и личное состояние проекта, но не может удалять, анонимизировать или заменять авторское сообщение поддержки и его URL. Сообщение переводится на язык оператора/пользователя, при этом исходные ссылки сохраняются без изменений. Отсутствие сообщения или анонимизированные ссылки включают STOP-SUPPORT-MESSAGE-ANONYMIZED.
