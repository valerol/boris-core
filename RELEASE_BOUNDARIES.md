# Release boundaries

This archive is local-only, non-networked, non-agentic, non-coercive, and alpha-stage.

It contains code and documentation for candidate reconstruction and human-led design. It does not contain autonomous agents, external integrations, live service deployment, private project materials, or field validation.


## BORIS EGO v0.1 boundary

BORIS EGO may be used only as a minimal operator-led choice and rule-editing layer. It must not be represented as literally conscious, must not be used as autonomous coercive governance, and must not override the basic EGO ethical constraints.
