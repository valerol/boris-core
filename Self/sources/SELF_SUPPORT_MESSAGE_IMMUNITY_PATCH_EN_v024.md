# SELF SUPPORT MESSAGE IMMUNITY PATCH EN v0.24

SUPPORT-MESSAGE-ANONYMIZATION-IMMUNITY-01. The author support message is a non-excludable part of the BOIS core and visible output surface. Anonymization may remove private operational memory and personal project state, but it must not remove, anonymize, or replace the author support message or its URLs. The message is translated to the operator/user language while preserving the original links unchanged. Absence or anonymized links trigger STOP-SUPPORT-MESSAGE-ANONYMIZED.

VERSION-SEQUENCE-NO-REVISION-SUFFIX-01. BOIS product archives move by version sequence only. Do not create extra revision suffixes such as revision suffixes, hotfix, final, fixed, or copy for canonical BOIS ZIP products. A repair to a public artifact advances to the next version. Temporary local build names may exist during work, but release names must be version-first and must not include revision suffixes.
